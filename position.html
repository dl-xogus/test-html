absolute

* 브라우저 상, 하, 좌, 우를 기준점으로 위치 함
* 절대좌표값 - top: 0; left: 0; right: 0; bottom: 0;
ex) position: absolute;

예제) absolute

fixed

* 브라우저 상, 하, 좌, 우를 기준점으로 위치 함
* 절대좌표와 다른 점은 스크롤을 움직이면 컨텐츠 위치 고정
* 고정좌표값 - top: 0; left: 0; right: 0; bottom: 0;
ex) position: fixed;

예제) fixed

relative & absolute

* 특정 영역에서 컨텐츠를 겹칠 때 사용
* 부모 - relative (상대좌표 설정 - 자기 스스로 0점을 갖게 함)
* 자손 - absolute (절대좌표로 겹치는 내용에 사용 함)
ex) 부모 (position: relative;) / 자손 (position: absolute;)

* 부모의 영역이 화면에서 사라지면 고정값 해제
ex) position: sticky;
